Udacity CS253 Final Project
Built a Mini Wiki website

The wiki website is http://jiajun-miniwiki.appspot.com

Main Features:
1. Built a fully functional Wiki website with Python by using Google App Engine.
2. Developed both frontend and backend based on Google NoSQL Datastore (DB), webapp2 framework and Jiaja2 Template engine.
3. Implemented user register/login, page create/view/edit and history version access/edit.
